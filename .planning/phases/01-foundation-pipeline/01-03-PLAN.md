Create transcribe/cli.py with main() function as Click command:

Command definition (per user decisions from CONTEXT.md):
- @click.command()
- @click.argument('video_file', type=click.Path(exists=True))
- @click.option('-q', '--quiet', is_flag=True, help='Suppress progress output')
- @click.option('--force', is_flag=True, help='Overwrite existing output without asking')

Function signature: def main(video_file, quiet, force):

Implementation flow:

1. Convert video_file to Path object

2. Run early validations (fail fast pattern from research):
   - validate_environment() - catches RuntimeError, prints error, sys.exit(1)
   - validate_video_file(video_path) - catches exceptions, prints error, sys.exit(1)

3. Construct output path (satisfies AUDIO-02):
   - output_path = video_path.parent / f"{video_path.stem}_transcript.md"
   - If exists and not force: use click.confirm(f"{output_path.name} exists. Overwrite?")
   - If user says no: click.echo("Cancelled."), return

4. Process video using Pattern 3 (two-stage pipeline with cleanup) from research:
   - Create temp audio file using tempfile.NamedTemporaryFile(suffix='.wav', delete=False)
   - Wrap in try/finally for cleanup
   - If not quiet: show progress using click.echo for simple text updates ("Extracting audio...", "Transcribing...", "Done.")
   - Extract audio: extract_audio(video_path, temp_audio)
   - Transcribe: transcriber = Transcriber(); segments, info = transcriber.transcribe(temp_audio)
   - Format: transcript_text = format_transcript(segments, info, video_path.name)
   - Write: output_path.write_text(transcript_text, encoding='utf-8')
   - Finally: Clean up temp_audio if exists

5. Success message:
   - click.echo(f"Transcript saved to: {output_path}")

Error handling:
- Catch all exceptions in main try block
- Print user-friendly error messages for known exceptions (FileNotFoundError, ValueError, RuntimeError)
- For unexpected exceptions: show error and suggest checking FFmpeg/file

Imports: click, pathlib.Path, tempfile, sys
From modules: validate_environment, validate_video_file, extract_audio, Transcriber, format_transcript

Use simple text progress (click.echo) rather than click.progressbar for Phase 1 simplicity (research Open Question 2 - indeterminate progress during transcription).

Entry point check at bottom:
```python
if __name__ == '__main__':
    main()
```
