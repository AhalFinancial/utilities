**transcribe/parallel.py — Parallel chunk transcription:**

Create a module with these components:

1. `_transcribe_single_chunk(chunk_path, model_size, language)` — MODULE-LEVEL function (not nested, not lambda — required for pickle/ProcessPoolExecutor):
   - Creates a fresh Transcriber(model_size) instance in the worker process
   - Calls transcriber.transcribe() with language parameter and VAD enabled
   - Returns `(segments_list, info)` tuple

2. `transcribe_chunks_parallel(chunks, model_size="small", language=None, max_workers=None, on_chunk_complete=None)` function:
   - `chunks` is list of `(chunk_id, chunk_path, start_time_offset, duration)` from chunker
   - Calculate max_workers if None: `min(os.cpu_count() or 2, max(1, available_ram // 2))` where available_ram is estimated at 8GB default (avoid psutil dependency; user can override with max_workers param)
   - Submit all chunks to ProcessPoolExecutor
   - Use `as_completed(futures)` to process results as they finish
   - For each completed chunk, call `on_chunk_complete(chunk_id)` callback if provided (this is how progress updates)
   - Adjust segment timestamps: add `start_time_offset` to each segment's `.start` and `.end`
   - Sort results by chunk_id to maintain order
   - Call `deduplicate_overlap()` between consecutive chunk texts
   - Return `(merged_segments, info, all_segment_data)` — merged_segments is a list of segment-like objects with adjusted timestamps

3. `_adjust_timestamps(segments, offset)` — helper to shift timestamps by offset

4. Define a simple `MergedSegment` dataclass/namedtuple with fields: `start`, `end`, `text`, `avg_logprob` — used for reassembled segments since original Whisper segment objects are not easily modified

**transcribe/progress.py — Two-stage progress display:**

Create a module with:

1. `ProgressDisplay` class:
   - `__init__(self, quiet=False)` — stores quiet flag
   - `start_extraction(self)` — prints "Extracting audio... " (no newline) if not quiet
   - `finish_extraction(self)` — prints "[done]" if not quiet
   - `start_transcription(self, total_chunks)` — creates tqdm progress bar if not quiet, stores it
     - Bar format: `'{desc}: {bar}| {n_fmt}/{total_fmt} chunks [{elapsed}<{remaining}]'`
     - desc="Transcribing"
     - If quiet: no-op
     - If total_chunks == 1: just print "Transcribing..." (no progress bar for single chunk)
   - `update_transcription(self, chunk_id=None)` — calls tqdm.update(1) if bar exists
   - `finish_transcription(self)` — closes tqdm bar if exists, or prints "Done." if quiet
   - `warn(self, message)` — prints warning to stderr using click.echo(err=True), respects quiet (suppress in quiet mode? No — warnings always show per user decision to "always save but warn")

2. Handle non-TTY: if `sys.stdout.isatty()` is False, disable tqdm (set `disable=True`)

**Updated processing flow:**

1. Validation phase (unchanged): validate_environment(), validate_video_file()
2. Output path construction (unchanged)
3. Overwrite check (unchanged)

4. **Stage 1 — Audio extraction with progress:**
   - Create ProgressDisplay(quiet=quiet)
   - progress.start_extraction()
   - extract_audio(video_path, temp_audio)
   - progress.finish_extraction()

5. **Stage 2 — Language detection:**
   - Create Transcriber(model_size="small")
   - Detect language: `language, lang_prob = transcriber.detect_language(temp_audio)`
   - If not quiet: `click.echo(f"Detected language: {language} ({lang_prob:.0%})")`

6. **Stage 3 — Chunking decision:**
   - Call `chunk_audio(temp_audio)` — returns chunk list
   - If single chunk (short video): use simple path
   - If multiple chunks (long video): use parallel path

7. **Stage 4a — Simple path (single chunk, short video):**
   - Call `transcriber.transcribe_with_quality(temp_audio, language=language)`
   - Get `(segments, info, confidence_pct)`

8. **Stage 4b — Parallel path (multiple chunks, long video):**
   - progress.start_transcription(total_chunks=len(chunks))
   - Call `transcribe_chunks_parallel(chunks, model_size="small", language=language, on_chunk_complete=lambda cid: progress.update_transcription(cid))`
   - progress.finish_transcription()
   - Validate quality on merged segments
   - If low confidence, warn and re-run with medium model (per user decision)

9. **Stage 5 — Format and save:**
   - Call `format_transcript(segments, info, video_path.name, confidence_pct=confidence_pct, model_name=transcriber.model_size)`
   - Write to output_path
   - If confidence_pct and confidence_pct < 50: `progress.warn(f"Low confidence detected ({confidence_pct:.0f}%)")`

10. **Cleanup:** Remove temp audio and chunk temp directory (use `cleanup_chunks()` from chunker)
