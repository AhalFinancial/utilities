  <action>
Create transcribe/formatter.py with two functions following research Pattern 4:

1. format_timestamp(seconds: float) -> str:
   - Convert seconds to HH:MM:SS format
   - hours = int(seconds // 3600)
   - minutes = int((seconds % 3600) // 60)
   - secs = int(seconds % 60)
   - Return f"{hours:02d}:{minutes:02d}:{secs:02d}"

2. format_transcript(segments, info, video_filename: str) -> str:
   - Build markdown with metadata header (per user decision from CONTEXT.md):
     - "# Transcript"
     - Source: {video_filename}
     - Date: {datetime.now().strftime('%Y-%m-%d %H:%M:%S')}
     - Duration: Calculate from segments or info.duration, format as "Xm Ys"
     - Language: {info.language} (confidence: {info.language_probability:.2%})
     - Model: "faster-whisper (small)"
   - Add separator: "---"
   - Iterate segments, format each as: "[{timestamp}] {segment.text.strip()}" where timestamp uses format_timestamp(segment.start)
   - Join with newlines, return complete markdown string

Imports: datetime from datetime module

Output language matches audio language per user decision (Spanish audio → Spanish text).
