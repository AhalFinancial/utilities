Create transcribe/validators.py with two validation functions following research patterns:

1. validate_environment():
   - Use shutil.which('ffmpeg') to check system FFmpeg
   - If None, raise RuntimeError with install instructions for macOS (brew install ffmpeg), Ubuntu (apt-get install ffmpeg), Windows (download from ffmpeg.org, add to PATH)
   - Per research: System FFmpeg is required for audio extraction (PyAV bundled with faster-whisper only handles transcription)

2. validate_video_file(video_path: Path):
   - Check path.exists() - raise FileNotFoundError if missing
   - Check path.is_file() - raise ValueError if not a file
   - Check extension against {'.mp4', '.mkv', '.webm', '.avi'} (case-insensitive via path.suffix.lower())
   - If invalid extension, raise ValueError with clear message listing supported formats
   - Satisfies AUDIO-01 requirement (all 4 formats validated)

Import: shutil, pathlib.Path

Add docstrings explaining when to call (startup for environment, before processing for file).
